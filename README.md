# KeepExperience

Keep experience when players die

# Installation
- Download jar file from [here.]()
- Put it in the `plugins` folder
- Reload server

# Commands

|     Commands    |        Permission        |       Description      |
|:---------------:|:------------------------:|:----------------------:|
| /keepexperience |    keepexperience.cmd    | Toggle keep experience |
